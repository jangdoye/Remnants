&nbsp;프로젝트 소개



Remnants는 감정을 테마로 한 퍼즐 어드벤처 게임입니다.

플레이어는 왜곡된 감정 공간을 탐험하며 감정적 단서를 회복해 나가는 구조로 구성되어 있습니다.



내가 담당한 기능

1\. 싱글톤 기반 상태 관리 시스템 (TeleportState)



씬 전환 후 플레이어 진행 상태 유지

로비 ↔ 퍼즐 방 간 이동 제어

방문 여부에 따라 연출/진행도 변화

&nbsp;

2\. 화면 연출 시스템



흑백 필터 적용

깨진 화면 UI 오버레이

특정 퍼즐 조건 달성 시 정상 화면 전환
